# SpaceWalk

Turn floor plan blueprints into interactive 3D walkthroughs for real estate and venue visualization.

## Details
- **URL**: https://spacewalk.georg.miami
- **Folder**: personal/spacewalk/
- **Stack**: React + TypeScript (Vite), Three.js, AI-rendered room views
- **Repo**: georgchimion-oss/spacewalk (public)

## Features
- Upload professional floor plan blueprints (with materials spec)
- Interactive 3D dollhouse-style overview (Three.js)
- Click rooms for AI-rendered walkthrough views
- Material/finish visualization from blueprint specs

## Design System
- Dark theme (#08080A bg, #1C1C23 surface)
- Fonts: Space Grotesk (headings), DM Sans (body), JetBrains Mono (code)
- Colors: #F0845C (warm/CTA), #00D2C8 (teal), #10B981 (green), #6366F1 (accent)

What level of 3D should the dollhouse view be?

This determines the technical approach and visual complexity

A

Styled 2.5D (Isometric)

Elevated floor plan with colored walls, room labels, and material swatches. Think "The Sims" build mode. No true 3D camera — fixed isometric perspective. Fast to build, works on any device.

┌─────────────┐
│ ╱ LIVING ╱ │ ← walls have height/shadow
│╱─────────╱ │ ← floor has material color
│ ┌────┐ │
│ │BATH│ │ ← rooms are labeled blocks
│ └────┘ │
└─────────────┘
Fixed angle, no rotation
B

True 3D Dollhouse (Three.js)

Fully rotatable 3D model with extruded walls, textured floors, and basic furniture silhouettes. Orbit controls to spin/zoom. Like a Matterport dollhouse but simpler. Medium complexity.

╱‾‾‾‾‾‾‾‾‾‾‾╲
│ LIVING │ ← 3D extruded walls
│ ▪ ▪ ▪ │ ← basic furniture shapes
│ ┌────┐ │ ← textured floors
│ │BATH│ │
╲___________╱
🔄 Orbit / zoom / pan
C

AI-Enhanced 3D

Same as B, but AI generates realistic textures for walls/floors based on the materials list (e.g., "walnut paneling" → actual wood texture). Higher wow factor, but depends on AI image generation quality and adds latency.

Same as B but with:
🎨 AI-generated wall textures
🪵 Realistic floor materials
💡 Ambient lighting per room
⏳ +5-15s generation time
